# Hero Section Images

This directory contains optimized images for the hero section.

## Image Optimization Process

The images in this directory have been optimized using the following process:

1. Converted to WebP format for better compression
2. Resized to appropriate dimensions for different screen sizes
3. Compressed to reduce file size by at least 30% while maintaining quality
4. Created fallback images in JPEG/PNG format for browser compatibility

## Available Images

- `optimized-image-1.webp` - Primary hero image in WebP format
- `optimized-image-2.webp` - Secondary hero image in WebP format
- `fallback-image-1.jpg` - Fallback for browsers that don't support WebP
- `fallback-image-2.jpg` - Fallback for browsers that don't support WebP
